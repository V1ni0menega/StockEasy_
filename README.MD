# StockEasy

Sistema de controle de estoque desenvolvido em Flutter para fins de aprendizado.

## 🚀 Tecnologias

- Flutter
- Dart
- SQLite
- Git/GitHub

## 📚 Objetivo

Projeto criado para praticar:

- CRUD
- Banco de dados local
- Interface com Flutter
- Organização de projeto
- Git e Conventional Commits

## 📂 Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Movimentações
- Tela de login
- Dashboard

## 🛠️ Como executar

```bash
flutter pub get
flutter run
```
