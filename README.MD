# StockEasy

Sistema de controle de estoque desenvolvido em Flutter para fins de aprendizado.

---

## 📚 Objetivo

O projeto StockEasy está sendo desenvolvido como parte do curso Técnico em TI, no módulo de Desenvolvimento de Aplicativos II.

O objetivo do projeto é praticar conceitos de desenvolvimento mobile utilizando Flutter, além de aplicar conhecimentos de:

- CRUD
- Banco de dados local
- Interface gráfica
- Estruturação de aplicações

Além do aprendizado técnico, o projeto também busca simular práticas utilizadas no mercado de desenvolvimento de software.

---

## 🚀 Tecnologias utilizadas

- Flutter
- Dart
- SQLite
- Git
- GitHub

---

## 🛠️ Funcionalidades

- Tela de login
- Cadastro de produtos
- Controle de estoque
- Movimentações
- Dashboard inicial
- Banco de dados local

---

## 🚀 Evolução do projeto

O projeto inicialmente foi proposto apenas como uma aplicação acadêmica para prática de CRUD.

Por iniciativa própria, o desenvolvimento foi expandido para incluir estudos de ferramentas e metodologias utilizadas profissionalmente no desenvolvimento de software, incluindo:

- Versionamento com Git
- Hospedagem e organização no GitHub
- Conventional Commits
- Uso de branches
- Organização de documentação
- Estruturação profissional do projeto

Com isso, o projeto passou a servir também como prática de engenharia de software e fluxo de desenvolvimento profissional.
